Instructions

This simple program should be ran in Node.js. To do this, first install Node.js. When you're done, open up a console/terminal, navigate to the folder /HW1_JSIntroduction/01.CurrentTime/ and run the command 'node current_time.js'. You should see the current date and time, neatly formated. :)

If you don't want to mess with Node, you can view the result in your browser's console. Just hit F12 and open the 'Console' tab.
